At RDV, we take data security seriously. Here's an overview of the security measures we implement to protect your robot data.

## Encryption

All data is encrypted both in transit and at rest:
- **TLS 1.3** for all network communications
- **AES-256** encryption for stored data
- **Zero-trust** architecture throughout our platform

## Access Control

We provide granular access control:
- Role-based access control (RBAC)
- Multi-factor authentication (MFA)
- IP whitelisting
- API key management

## Audit Logging

Every action is logged:
- User authentication events
- Data access and modifications
- Configuration changes
- API calls

## Compliance

Our platform is designed to meet industry standards:
- SOC 2 Type II certified
- GDPR compliant
- ISO 27001 certified

Your data security is our top priority.
