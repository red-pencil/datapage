We are proud to announce the launch of our new API platform, designed to make integrating robot data into your applications easier than ever.

## Key Features

- **RESTful Architecture**: Industry-standard REST API design
- **Comprehensive Documentation**: Clear and detailed API references
- **Multiple Authentication Methods**: API keys, OAuth 2.0 support
- **Rate Limiting**: Fair usage policies with generous limits
- **Real-time Streaming**: WebSocket support for live data

## Getting Started

Our new API makes it simple to:
1. Fetch robot telemetry data
2. Query historical data
3. Trigger robot actions
4. Subscribe to real-time events
5. Manage robot fleets

Visit our documentation to get started today!

## API Endpoints

```
GET /api/v1/robots
GET /api/v1/robots/{id}/telemetry
POST /api/v1/robots/{id}/commands
GET /api/v1/events
```

Stay tuned for more API enhancements in the coming weeks!
